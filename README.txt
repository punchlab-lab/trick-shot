TRICK SHOT — see yourself on a delay
=========================================

HOW TO USE
1. Double-click  "Start Delayed Camera.command"
2. In the browser that opens, click  Start  and allow the camera.
3. Move — you'll see yourself play back a few seconds later.

To quit: close the Terminal window that opened.

CONTROLS
  + / −  buttons (or the ↑ / ↓ arrow keys) ... change the delay (0.5–10s, default 3s)
  SPACE  key, or the "● Clip" button ........ capture the last few seconds
  C  key, or the "☰ Clips" button ........... show / hide the clips panel
  F  key, or the "Full" button .............. full screen
  Camera menu (only if you have more than one) ... pick a camera

CLIPS
  Press SPACE to capture what just happened — roughly the last 2× your delay
  (so ~6 seconds at the default 3s). It takes a few seconds; you'll see
  "Saving…" then "Clip saved ✓".

  Captured clips appear in the CLIPS panel on the left (open it with the
  "☰ Clips" button or the C key):
    • Click a clip  → plays full screen on a loop. Press the ✕ (or Esc) to
                      go back to the live delayed view.
    • ⤓ on a clip   → saves that clip to your Downloads to keep it.
    • 🗑 on a clip  → removes it from the panel.

  NOTE: the panel is cleared when you reload or close the app. Anything you
  want to keep, download it (⤓) first. Downloaded files are named like
  delayed-clip-20260624-153012.mp4 — double-click to play. (If your browser
  can only make WebM, open it with Chrome or VLC.)

FIRST TIME ONLY
If macOS blocks the launcher with a security warning, right-click
"Start Delayed Camera.command" → Open → Open. You only do this once.

Nothing is recorded or sent anywhere — it all runs on your Mac.
